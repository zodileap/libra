# 架构草图（v0）

## 分层

- 平台层：账号、授权、订阅/购买、企业管理、智能体入口。
- 智能体层：代码智能体、三维模型智能体（可单独售卖）。
- Core 层：流程管理、激活码鉴权、大模型调用网关。
- 基础设施层：Entity（DAO）、API、Sandbox、日志监控。

## 模块边界

- 平台与智能体解耦：平台只做入口与治理，不内嵌特定智能体业务细节。
- 智能体模块可单独启停、授权、计费、发布。
- Core 不承载产品页面，只承载跨模块复用能力与统一协议。

## 首发智能体

- 代码智能体：对话式生成 + Web 预览（后端 sandbox 支持）。
- 三维模型智能体：Web 展示 + Desktop 对接 Blender/ZBrush。
