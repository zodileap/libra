# Schema Review Checklist

## 实体与字段

- 每个实体是否有明确业务职责。
- 每个字段是否有类型、含义、可空性和默认值说明。
- 是否区分了主键、业务唯一键和外键关系。

## 索引与约束

- 高频查询是否有对应索引。
- 唯一性、状态约束和数据完整性是否有结构级保证。
- 是否说明了冗余字段或反范式设计的理由。

## 迁移与回滚

- 是否定义了 migration 顺序、历史兼容和数据回填。
- 回滚时哪些步骤可逆、哪些步骤不可逆。
- 是否标注线上风险、锁表风险或长事务风险。

## 一致性风险

- 幂等、并发写入、事务边界是否明确。
- 级联删除 / 更新是否安全。
- 是否存在脏数据、重复数据或历史格式兼容风险。
