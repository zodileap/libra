# P0 Light/Dark 手工验收清单（Desktop）

## 1. 验收前准备

- 启动命令：`pnpm -C apps/desktop dev`
- 验收尺寸：`1440x920`、`1280x900`、`1024x900`
- 主题模式：`Light`、`Dark`

## 2. 页面清单

- [ ] 登录页：`/login`
- [ ] 首页：`/home`
- [ ] 会话页：`/session/:sessionId`（任意会话）
- [ ] 设置页：`/settings/general`

## 3. 每页检查点（Light 与 Dark 都要过）

- [ ] 文本颜色可读，无对比度明显不足区域。
- [ ] 卡片边框、阴影、背景层次自然，无突兀硬边。
- [ ] 侧边栏与主内容区分层清晰，滚动时无明显抖动。
- [ ] 输入区、按钮、列表行 hover/active 状态符合主题。
- [ ] 窄窗口下（1024x900）无内容截断、重叠、溢出。

## 4. 会话页附加检查

- [ ] 会话消息区滚动时，底部输入区位置稳定。
- [ ] 消息区出现滚动条时，内容宽度不跳动。
- [ ] 操作提示卡片（warning/info/danger）颜色层次符合主题。
